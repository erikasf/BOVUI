I've coded the bov UI Elements to be very flexible and powerful, as such, some of them are kind of complicated. Please see the floating navigation to navigate the docs (you can drag it around or minimize it to get it out of your way).

If you're using more than one style of an element type, you may find it beneficial to combine some CSS. I've coded it so that it's really simple to just grab the CSS you need, but by all means, compact it if you can/want.

Some features in this kit require jQuery, the jQuery Uniform plugin, and/or the jQuery UI plugin. Information on which items require these elements are found in the element's individual docs. Unfortunately, I cannot provide additional customization, but I've coded each element to provide the most flexibility. You can certainly add or adjust the code as needed. I'll try to explain how each set of code works so that you can fiddle on your own.

Menus

There are two basic menu styles, light and dark. You'll need to grab the HTML and the CSS.

  1. Grab the HTML. Make sure the first li has a class of "first" and the last li has a class of "last". You don't need to have a search bar in there, just remove that whole li and it should look just fine.
    <div id="menu1">
	<ul>
	<li class="first"><a href="#">Home</a></li>
	<li><a href="#">Products</a></li>
	<li><a href="#">Company</a></li>
	<li><a href="#">Portfolio</a></li>
	<li><a href="#">Contact Us</a></li>
	<li class="last"><form><input type="text" value="Site Search" onblur="if (this.value == ''){this.value = 'Site Search';}" onfocus="if (this.value == 'Site Search') {this.value = '';}" /></form></li>
	</ul>
</div>

Striped Progress Bars

The striped progress bars are fairly simple to implement. They're basic divs with spans that can be whatever width % you'd like. We have the width set in the CSS, but you can remove that and set it inline or with javascript if you'd like. Here's what you do:

  1. Grab the HTML. Class progress1 is the dark stripe bar and class progress2 is the blue bar. 
    <div class="progress1"><span class="status"></span></div>
<div class="progress2"><span class="status"></span></div>
  2. Then grab the associated CSS: 
    /* Grey Progress Bar */
.progress1 {width: 300px;height: 17px;display: block;position: relative;background: url(../images/bov-ui-sprite.png) 0 -715px;}

Sliders

Sliders that you can drag and pull require jQuery and the jQuery UI plugin. You can find more details on that here: jQuery UI. The basics of what you need are included in the download.

Both types of sliders (single and double) use the jQuery UI plugin to enable the sliding and then some custom jQuery magic to push values into hidden inputs so that you can then do something with the data you collect if you need to. As currently written, they'll collect what percentage of the full width of the bar is selected. For the double sliders, they collect two sets of information and add them separately to two hidden inputs, the percentage value of the first bar and the percentage value of the second bar. You can then manipulate that data however you choose.

Here are the steps.

  1. Grab the HTML: 
    <div id="slider1"><div class="sliderstatus"></div><input type="hidden" class="value" /></div>
<div id="slider2"><div class="sliderstatus"></div><input type="hidden" class="value" /></div>
<div id="slider3">
	<div class="sliderstatus"><input type="hidden" class="value" /></div>
	<div id="slider3start">
		<div class="sliderstatusstart"><input type="hidden" class="valuestart" /></div>
	</div>
</div>
<div id="slider4">
	<div class="sliderstatus"><input type="hidden" class="value" /></div>
	<div id="slider4start">
		<div class="sliderstatusstart"><input type="hidden" class="valuestart" /></div>
	</div>
</div>
  2. Link to both jQuery and jQuery-UI in the head section. If you choose to link to your own copy of jQuery - update the src and also make sure the src to your copy of the jQuery uniform plugin is correct as well. 
    <script type='text/javascript' src='http://ajax.googleapis.com/ajax/libs/jquery/1.5.2/jquery.min.js'></script>
<script type='text/javascript' src='js/jquery-ui-1.8.2.custom.min.js'></script>

Dropdown, File, Radio, and Checkbox Elements

The dropdown, file selector, radio, and checkboxes are styling using jQuery's Uniform Plugin. Basic needs are included in the download, and you can find full documentation on the plugin here: jQuery Uniform Plugin. Basically, this plugin uses a sprite to style almost everything, so we adjusted their sprite template to match our styles. Included in the download is our sprite PSD so you can adjust everything for yourself further. To use these styles, complete the following steps:

  1. Link to jQuery, jQuery Uniform plugin, and Uniform's stylesheet in the head section of your pages. Make sure you're using proper relative urls or use the full url to where you've uploaded the files: 
    <link rel="stylesheet" href="css/uniform.default.css" type="text/css" media="screen" />
<script type='text/javascript' src='http://ajax.googleapis.com/ajax/libs/jquery/1.5.2/jquery.min.js'></script>
<script type='text/javascript' src='js/jquery.uniform.min.js'></script>
  2. Create your forms however you want them. Be sure to add a class to them if you want to differentiate the style of only these items and not all form items on your page. For example, on this demo, we don't want this styling to apply to the quantity input or to our switch checkboxes, so we put a class of bov-ui on the items we do want styled so we can tell jQuery/Uniform that we only want those items styled. Here's an example: 
    <select class="bov-ui">
<option>One</option>
<option>Two</option>
<option>Three</option>
</select>

<input type="file" class="bov-ui"/>
<input type="radio" name="number" value="1" class="bov-ui"/>
<input type="radio" name="number" value="2" checked="checked" class="bov-ui"/>
<input type="checkbox" name="number" value="1" class="bov-ui"/>
  3. Now, you have to initiate the Uniform plugin. We did this within the document ready function and had it apply to every select on the page, every file input, and any input with a class of bov-ui. See below: 
    $('select, input:file, input.bov-ui').uniform();

Quantity Input

The quantity input is a fieldset that includes the quantity input and some spans that are then triggered with jQuery to increase or decrease the value of the input.

  1. Make sure you're linking to jQuery: 
    <script type='text/javascript' src='http://ajax.googleapis.com/ajax/libs/jquery/1.5.2/jquery.min.js'></script>
  2. Add the HTML: 
    <fieldset class="bov-ui-value">
		<input type="text" value="1" />
		<span class="value-inc" onclick="increaseValue('.bov-ui-value');"></span>
		<span class="value-dec" onclick="decreaseValue('.bov-ui-value');"></span>
	</fieldset>
  3. Add the CSS: 
    /* Quantity Input */
.bov-ui-value {position: relative;width: 50px;}
.bov-ui-value input{border-style: none; width: 30px;padding: 3px 0px 3px 10px;height: 20px;background: url(../images/bov-ui-sprite2.png) -415px -178px;position: relative;font: bold 13px/24px Arial, Helvetica, Geneva, sans-serif;color: #908964;text-shadow: #fff 0 2px 0;}
.bov-ui-value input:hover {background-position: -415px -211px;}
.bov-ui-value span {display: block; position: absolute;cursor: pointer;height: 13px;width: 14px;top: 0px;right: 0px;background: url(../images/bov-ui-sprite2.png) -547px -178px;}
.bov-ui-value span.value-inc:hover {background-position: -547px -211px;}
.bov-ui-value span.value-dec {top: 13px;background-position: -547px -191px;}
.bov-ui-value span.value-dec:hover {background-position: -547px -224px;}
  4. Enable the two spans to increase and decrease. Add the following javascript to the head section of your page or within your javascript functions document. It does not need to go inside the document ready function. 
    var increaseValue = function (itemId) {
	var initialValue = $(itemId).find('input').attr('value');
	var newValue = parseInt(initialValue) + 1;
	$(itemId).find('input').attr('value', newValue);
};
var decreaseValue = function (itemId) {
	var initialValue = $(itemId).find('input').attr('value');
	var newValue = parseInt(initialValue) - 1;
	$(itemId).find('input').attr('value', newValue);
};

Switches

Both the on/off switches and the left/right smaller switches are placed into the HTML as checkboxes, and then hidden and displayed as divs and spans to create the effect on this demo. You'll need to put the checkbox into the HTML and then manipulate it using some jQuery script in order to get the look seen. Here's what you do:

  1. Make sure you're calling jQuery in the head of your page: 
    <script type='text/javascript' src='http://ajax.googleapis.com/ajax/libs/jquery/1.5.2/jquery.min.js'></script>
  2. Use your standard checkbox HTML, making sure to add a specific ID or class that you'll use to designate which items to change later: 
    <input type="checkbox" name="switch1" class="onoffswitch"/>
<input type="checkbox" name="switch3" class="leftrightswitch"/>
  3. Add the following CSS to your CSS file or section: 
    /* On/Off Switch */
.onoffswitch {width: 67px; height: 27px;position: relative;overflow: hidden; display: block;cursor: pointer;}
.onoffswitch .background {width: 145px; height: 27px;display: block;position: absolute; top: 0; left: 0;background: url(../images/bov-ui-sprite.png) -414px -831px; z-index: 1;}

/* Left/Right Switch */
.leftrightswitch {width: 32px; height: 17px;position: relative;overflow: hidden; display: block;cursor: pointer;}
.leftrightswitch .background {width: 72px; height: 17px;display: block;position: absolute; top: 0; left: 0;background: url(../images/bov-ui-sprite.png) -424px -811px; z-index: 1;}
  4. Next you'll want to tell jQuery what to do with that HTML. For the on/off switch, add this to your document ready function: 
    // Iterate over checkboxes
$("input:checkbox.onoffswitch").each(function() {
  // Insert mark-up for switch
  $(this).before(
    '<span class="onoffswitch">' +
    '<span class="background" />' +
    '</span>'
  );
  // Hide checkbox
  $(this).hide();
    
  // Set inital state
   if (!$(this)[0].checked) { // if not checked (off)
        $(this).prev().find(".background").css({left: "-78px"});
   }
}); // End each()
      
  // Toggle switch when clicked

  // If on, slide switch off
